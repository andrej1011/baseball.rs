_Final Project for Software Lab 2: Web Design_

**Developed by** Andrej Rajkov

## ‚öæÔ∏è Project Overview

**Baseball .rs** was conceived as the central online hub for the Serbian baseball community. Since baseball in Serbia operates largely underground, the goal of this website was to provide a professional, accessible, and comprehensive resource for both dedicated fans and new participants.

The project fulfilled the requirements for the "Software Lab 2 - Web Design" course, demonstrating expertise in responsive design, information architecture, and modern front-end development techniques.

## üèüÔ∏è Key Content and Features

The website structure was designed to serve various community needs:

-   **Comprehensive Tutorials:** Detailed guides for beginners covering basic rules, positions, and equipment.
    
-   **History & Culture:** A dedicated section detailing the global history of baseball and the specific, unique history of baseball within Serbia.
    
-   **Club Directory:** Essential contact information and locations for all active baseball clubs in the country.
    
-   **Live Game Access:** A curated section where users could find links and resources to watch baseball games for free.
    

## üíª Technology Stack

This project was built using standard, modern web technologies:

-   **HTML5:** Structured, semantic markup for all content.
    
-   **CSS3 (Sass/SCSS):** Used for pre-processing, allowing for modular, maintainable, and highly responsive styling.
    
-   **JavaScript (Vanilla JS):** Implemented for interactive elements, navigation logic, and dynamic content manipulation (e.g., hamburger menus, carousels).
    
-   **Responsive Design:** The layout is fully responsive, utilizing CSS Grid and Flexbox to ensure optimal viewing across mobile, tablet, and desktop devices.
